21. 단위

- px : 사용자 접근성에 고려하면 사용하지 않는 것 추천

- % : 부모 요소를 기준으로 하는 백분율

- vw, vh : 뷰포트(화면) 넓이(w), 높이(h)를 기준으로 하는 백분율 단위입니다.
	1vw = 화면의 1%

<p style="width: 100px;">안녕하세요
    <strong style="width: 40vw">안녕</strong>  <!-- 부모가 px이기에 자식을 넓혀도 배경이 변하지 않는다-->
    </p>

- vmin, vmax : 화면 넓이 중 작은 값 또는 큰 값을 기준으로 하는 백분율

- em : 부모요소로 상속받은 요소의 글자크기를 기준으로 하는 배수 단위

p {
            background-color: aqua;
            font-size: 2em;
        }

        strong {
            font-size: 0.5em;  //  2em의 반값인 16px로 나옴
        }

* 기본 브라우저의 기본 px :16px

- rem : 최상위 요소의 글자 크기를 기준으로 하는 배수

- em과 rem을 사용하는 이유: 수정이 용이하기 때문, 배수형으로 훨씬 편하게 할 수 있음

22. 텍스트와 관련된 태그들

- h1 태그: 제목 태그 -> 목차이기에, 보다 정보를 훨씬 편리하게 가져갈 수 있음

* SEO : 접근성

* h1 태그는 한 페이지에 한 번만 사용을 권장

* 스타일을 목적으로 h1 태그를 쓰면 안 됨

* h1 ~ h6 순서대로 작성할 것


-  a 태그: 노션 참고

- p 태그: HTML의 문단의 역할을 수행

- br 태그: 줄바꿈 태그 -> 어지간하면 CSS 로 사용

- wbr 태그: 기준으로 줄바꿈이 일어남
               -> 한국어를 쓰고싶으면 word-break 코드 필요(한국어와 영어가 조금씩 다름)

- code : 코드 한 조각을 나타낼때 사용

- pre 태그 : HTML에 작성한 내용을 그대로 가져옴(공백, 고정폭 그대로 유지) -> firgure 태그로 접근서 올림

- strong 태그 : 기본적으로 굵은 글씨, 적극 권장

- em 태그 : 강한 어조로 말하게 해주는 글씨, 기본적으로 기울임 꼴로 적용, 한국에서는 잘 사용 안함

- q 태그 : 출처 + 짧은 인용문을 나타냄

- blockquote 태그 : 긴 인용문을 나타냄

- cite 태그 : 
<q cite="https://www.imdb.com/title/tt0062622/quotes/qt0396921">
		I'm sorry, Dave. I'm afraid I can't do that.
	</q>

<cite>— 어린왕자 여우의 말 중</cite>

- adress : 연락처 정보를 작성하는 태그, 이름을 반드시 포함 해야함


- mark : 하이라이트 텍스트 정의
	<mark> </mark>

- abbr 태그 : 약어임을 알려주는 태그 + title을 같이 씀

- dfn 태그 : 현재 맥락이나 문장에서 정의하는 용어 나타냄/ 주석이나 책 용어를 나타낼때 사용

- sup 태그 : 단어 위에 서수와 같은 작은 텍스트를 달아주는 태그

- sub 태그 : 단어 아래에 서수와 같은 작은 텍스트를 달아주는 태그

- kbd 태그 : 키보드 입력을 나타낼때 사용



23. 텍스트 꾸미기 

- color 색상: 폰트의 색상을 나타냄

- currentColor : 부모에 color 값이 있다면 상속으로 처리됨

- font-family : 글꼴 종류 -> 작성된 폰트중 있는 것을 순차적으로 적용

- 폰트 적용방법 : link, import / link 추천 또는 cdn 추천

- font-size : 글꼴 크기

- font-weight : 텍스트 굵기 설정 (bold, lighter, bolder, normal) 디테일 값: 100 ~ 900, 해당 디테일 값을 제공해야 가능함

- text-transform : 텍스트를 바꿈(대문자, 소문자) : uppercase , lowercase

- text-decoration : 텍스트를 꾸밀때 사용(밑줄, 오버라인, 취소선...)
